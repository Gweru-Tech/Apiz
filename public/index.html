üì∫ YouTube Endpoints

GET Download MP3
Extract and download audio from YouTube videos in MP3 format
/api/youtube/mp3?url=VIDEO_URL
Parameters:
url - YouTube video URL (required)
curl -X GET "https://your-api.onrender.com/api/youtube/mp3?url=https://youtube.com/watch?v=dQw4w9WgXcQ"
GET Download MP4
Download YouTube videos in MP4 format with quality selection
/api/youtube/mp4?url=VIDEO_URL&quality=highest
Parameters:
url - YouTube video URL (required)
quality - Video quality: highest/lowest (optional)
curl -X GET "https://your-api.onrender.com/api/youtube/mp4?url=https://youtube.com/watch?v=dQw4w9WgXcQ&quality=highest"
GET Search YouTube
Search for YouTube videos by keyword
/api/youtube/search?q=QUERY&limit=10
Parameters:
q - Search query (required)
limit - Number of results (optional, default: 10)
curl -X GET "https://your-api.onrender.com/api/youtube/search?q=music&limit=10"
GET Get Video Info
Get detailed information about a YouTube video
/api/youtube/info?url=VIDEO_URL
Parameters:
url - YouTube video URL (required)
curl -X GET "https://your-api.onrender.com/api/youtube/info?url=https://youtube.com/watch?v=dQw4w9WgXcQ"

ü§ñ GPT AI Endpoints

POST Chat with GPT
Send messages to GPT AI and receive intelligent responses
/api/gpt/chat
Request Body (JSON):
message - Your message to GPT (required)
conversationId - Conversation ID for context (optional)
model - GPT model (optional, default: gpt-3.5-turbo)
maxTokens - Max response length (optional, default: 1000)
curl -X POST "https://your-api.onrender.com/api/gpt/chat" \ -H "Content-Type: application/json" \ -d '{ "message": "Hello, how are you?", "conversationId": "12345" }'
GET Get Conversation History
Retrieve the history of a conversation
/api/gpt/conversation?conversationId=ID
Parameters:
conversationId - Conversation ID (required)
curl -X GET "https://your-api.onrender.com/api/gpt/conversation?conversationId=12345"
POST Clear Conversation
Clear the history of a conversation
/api/gpt/clear
Request Body (JSON):
conversationId - Conversation ID (required)
curl -X POST "https://your-api.onrender.com/api/gpt/clear" \ -H "Content-Type: application/json" \ -d '{"conversationId": "12345"}'

üöÄ Quick Start

JavaScript Example
// Download YouTube MP3 fetch('https://your-api.onrender.com/api/youtube/mp3?url=VIDEO_URL') .then(response => response.json()) .then(data => console.log(data)) .catch(error => console.error('Error:', error)); // Chat with GPT fetch('https://your-api.onrender.com/api/gpt/chat', { method: 'POST', headers: { 'Content-Type': 'application/json', }, body: JSON.stringify({ message: 'Hello!', conversationId: '12345' }) }) .then(response => response.json()) .then(data => console.log(data)) .catch(error => console.error('Error:', error));
Python Example
import requests # Download YouTube MP3 response = requests.get( 'https://your-api.onrender.com/api/youtube/mp3', params={'url': 'VIDEO_URL'} ) print(response.json()) # Chat with GPT response = requests.post( 'https://your-api.onrender.com/api/gpt/chat', json={ 'message': 'Hello!', 'conversationId': '12345' } ) print(response.json())
